## Requisitos:

* JDK 18 ó superior.
* MySQL server 5.1 ó superior.
* Librerias de Maven.

***

## Instalación:
* Compile el proyecto con el IDE de su preferencia.
* Ejecute desde la consola; en la raíz del proyecto, java Ejecutable.jar
